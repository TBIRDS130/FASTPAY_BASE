FastPay API

Secure backend for device sync, messaging, notifications, contacts, and admin workflows. Use this page as a quick reference for the most common endpoints and payloads.

Sync Contract

Client contract and pagination rules for sync clients.

GET /api/sync/contract/

Sync Status

Fetch or update device sync status for dashboards or mobile clients.

GET /api/sync/status/?device_id=ANDROID_ID

Device Registration

Register or update device metadata for the dashboard.

POST /api/devices/

Messages Sync

Bulk upload SMS messages from device.

POST /api/messages/

Notifications Sync

Bulk upload notification events from device.

POST /api/notifications/

Contacts Sync

Bulk upload contacts or upsert from device.

POST /api/contacts/

Dashboard Login

Authenticate dashboard users.

POST /api/dashboard-login/

Activity Logs

Retrieve admin activity history.

GET /api/dashboard-activity-logs/

Capture Ingestion

Browser extension capture endpoint.

POST /api/captures/

Gmail / Drive

Google integrations for email and file access.

GET /api/gmail/status/

Command Logs

Track device commands executed by APK.

POST /api/command-logs/

Admin Panel

Django admin interface for models and audit.

GET /admin/
API base: /api/
